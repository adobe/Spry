Sliding Panels Sample 2

This example shows an alternate application and styling of the Sliding Panels widget. In the example below, the view port for the SlidingPanels widget is made large enough to show 2 content panels. ID attributes are placed on specific content panels so that links can be used to slide the widget to the next set of panels to show.
