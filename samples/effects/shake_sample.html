Duration

The Shake effect allow you to specify a value for the "duration" through its constructor options which will increase the number of times the element will move both sides. The value is set in milliseconds.

The div element in this example will shake for 2 seconds. 

<script type="text/javascript">
   var second_example = new Spry.Effect.Shake('example2', {duration: 2000});
</script>
