Loading Tabbed Panels with the updateContent Util function

This sample shows how Tabbed Panels can load content dynamically, using the Spry.Utils.UpdateContent function in SpryData.js. When the tab is clicked, the HTML fragment is loaded into the Panel content DIV. The panels are each given an ID so we can direct the content to the panel.
